🚀 Getting Started with SOI Hub

Welcome to SOI Hub Admin

This guide will help you get started with the SOI Hub volunteer management system.

📊 Dashboard Overview

The main dashboard provides real-time statistics and quick access to key functions.

  • View volunteer statistics
  • Monitor event progress
  • Check system health
  • Access recent activity

🧭 Navigation

Use the main navigation menu to access different sections of the system.

  • Accounts: User management
  • Volunteers: EOI and profiles
  • Events: Events, venues, roles
  • Tasks: Task management
  • Reporting: Analytics and reports

👥 User Roles

Different user types have different permissions and access levels.

  • Admin: Full system access
  • Staff: General admin functions
  • VMT: Volunteer management
  • CVT: Corporate volunteers
  • GOC: Games operations

⚡ Quick Actions

Common tasks you can perform quickly from the dashboard.

  • Add new volunteers
  • Create events
  • Assign roles
  • Generate reports
  • View help documentation
