Creating New Events

Event Creation Process

  1. Navigate to Events in the admin panel
  2. Click "Add Event" button
  3. Fill in basic event information:
    • Event name and description
    • Event type (Competition, Training, Ceremony, etc.)
    • Start and end dates/times
    • Primary venue
  4. Configure event settings and requirements
  5. Set volunteer requirements and roles needed
  6. Save and publish the event

Common Issues & Solutions

Q: Event not showing up for volunteer registration

A: Check that the event status is "Published" and registration is "Open". Verify the event date is in the future.

Q: Volunteer assignments not saving

A: Ensure the volunteer meets the role requirements and the role hasn't reached capacity. Check for conflicting assignments.

Q: How to handle event cancellation

A: Change event status to "Cancelled", notify all registered volunteers, and update any dependent events or bookings.

Q: Venue double-booking issues

A: Use the venue calendar to check availability before creating events. The system will warn about conflicts.
