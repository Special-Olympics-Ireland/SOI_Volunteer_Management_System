üîó Use Case 6: JustGo Integration - Credential Validation

Scenario: Staff needs to validate volunteer credentials using JustGo API before assigning them to roles

  1. Staff searches for volunteer by MID using JustGo integration
  2. System retrieves member data via client.find_member_by_mid("SOI123456")
  3. System fetches current credentials via client.get_member_credentials(member_id)
  4. System validates credentials against role requirements
  5. If qualified, volunteer is assigned to role via POST /api/v1/events/assignments/
  6. If missing credentials, system generates training recommendations
  7. All JustGo interactions are logged for audit purposes
# JustGo Integration - Credential Validation Workflow from integrations.justgo import JustGoAPIClient # Step 1-2: Find member by MID client = JustGoAPIClient() member_data = client.find_member_by_mid("SOI123456") # Step 3: Get member credentials credentials = client.get_member_credentials(member_data['member_id']) # Step 4: Validate for specific role role_requirements = { 'required_credentials': ['Safeguarding', 'First Aid', 'Sports Coaching'], 'minimum_experience_years': 1, 'required_membership_types': ['VOLUNTEER', 'COACH'] } validation_result = client.validate_member_credentials_for_role( member_id=member_data['member_id'], role_requirements=role_requirements ) # Response shows qualification status { "is_qualified": true, "credentials": [ { "credential_id": "SAFE2024001", "credential_type": "Safeguarding", "status": "ACTIVE", "expiry_date": "2027-01-15" } ], "missing_requirements": [], "warnings": [ { "type": "EXPIRY_WARNING", "message": "First Aid expires in 45 days" } ] } # Step 5: If qualified, create assignment curl -X POST http://localhost:8000/api/v1/events/assignments/ \ -H "Content-Type: application/json" \ -H "Authorization: Bearer YOUR_JWT_TOKEN" \ -d '{ "volunteer": "volunteer_uuid_here", "role": "athletics_coach_role_uuid", "event": "event_uuid_here", "justgo_validation": { "member_id": "JG123456", "validated_at": "2026-04-15T10:30:00Z", "credentials_verified": true } }'
